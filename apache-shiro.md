### Apache Shiro
| 英文 | 中文 | 解释 |
| --- | --- | --- |
| Authentication | 认证 | 验证用户身份 |
| Authorization | 授权 | 访问控制 |
| Subject | 主题，科目 | 用户或者系统对象 |
| Principal | 主要的；主角，当事人 | 用户名等，被定义为一个系统用户 |
| Credential | 凭据，凭证 | 密码密钥等 |
| Permission | 权限，允许  | 权限 |
